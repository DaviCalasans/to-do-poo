# 📄 To-Do List - HTML, CSS & JavaScript

Este projeto consiste em uma aplicação web desenvolvida com **HTML, CSS e JavaScript puro**, com o objetivo de gerenciar tarefas de forma simples e dinâmica.  
O usuário pode cadastrar tarefas informando **nome, data e status**, com validações de formulário e persistência de dados no navegador utilizando **localStorage**, sem necessidade de recarregar a página.

---

# 🚀 Funcionalidades

✅ Cadastro de novas tarefas via formulário  
✅ Validação dos campos do formulário  
✅ Persistência de tarefas utilizando **localStorage**  
✅ Atualização dinâmica da interface com JavaScript  
✅ Busca e filtragem de tarefas 
---

# 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- LocalStorage API
